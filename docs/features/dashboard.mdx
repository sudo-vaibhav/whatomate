## Overview

The Dashboard provides a comprehensive view of your WhatsApp business operations. Monitor key metrics, track conversations, and stay on top of your messaging activities.

## Key Metrics

<CardGroup cols={2}>
  <Card title="Total Messages">
    Track the total number of messages sent and received across all your WhatsApp accounts.
  </Card>
  <Card title="Active Conversations">
    Monitor ongoing conversations with your customers in real-time.
  </Card>
  <Card title="Campaign Performance">
    View the success rate and engagement metrics of your bulk messaging campaigns.
  </Card>
  <Card title="Chatbot Analytics">
    Analyze how your automated responses are performing.
  </Card>
</CardGroup>

## Features

### Real-time Updates
The dashboard updates in real-time using WebSocket connections, ensuring you always see the latest data without refreshing.

### Time Range Filters
Filter your metrics by different time ranges:
- Today
- Last 7 days
- Last 30 days
- Custom range

### Quick Actions
Access frequently used actions directly from the dashboard:
- Send new message
- Create campaign
- View recent conversations
- Access settings
